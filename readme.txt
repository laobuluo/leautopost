== Description ==

LeAutoPost，一款可以实现自动将WordPress草稿文章定时发布的插件。公众号：老蒋朋友圈。

## 主要功能

* 定时将草稿文章发布。
* 可以设置外部激活定时。
* 设置过滤关键字，以及过滤部分内容格式，比如图片、URL等。
* 插件安装和介绍：https://www.lezaiyun.com/854.html

== Installation ==

* 1、把LeAutoPost文件夹上传到/wp-content/plugins/目录下<br />
* 2、在后台插件列表中激活LeAutoPost<br />
* 3、在左侧 设置 - LeAutoPost，菜单中设置信息。<br />
