# LeAutoPost

一款可以实现自动将 WordPress 草稿文章定时发布的插件。

**作者**：老蒋和他的小伙伴 | **公众号**：老蒋朋友圈

---

## 功能特性

### 核心功能

- **定时自动发布**：按设定间隔自动将草稿文章发布为正式文章
- **按时间排序**：优先发布最早创建的草稿
- **手动立即发布**：支持在后台一键立即发布一篇草稿

### 发布控制

| 功能 | 说明 |
|------|------|
| 发布频率 | 设置自动发布的时间间隔（秒），默认 3600 秒（1 小时） |
| 时间限制 | 仅在指定时间段内允许发布（如 09:00 - 22:00） |
| 发布数量限制 | 限制每日/周期内最大发布数量，留空表示不限制 |

### 内容过滤

发布前可对文章内容进行过滤处理：

- **关键词过滤**：每行一个关键词，发布时替换或删除
- **替换字符**：用于替换被过滤关键词的字符，留空则直接删除
- **过滤图片**：移除文章中的所有 `<img>` 标签
- **过滤链接**：移除文章中的所有 `<a>` 链接标签
- **过滤 URL**：移除文章中的纯文本 URL 地址

### 定时任务方式

- **WordPress 内置 Cron**：使用 WP 自带的定时任务（依赖站点访问触发）
- **外部 Cron（推荐）**：通过系统 Cron 或第三方定时服务访问专用 URL，提高发布可靠性

---

## 安装

1. 将 `leautopost` 文件夹上传到 `/wp-content/plugins/` 目录
2. 在 WordPress 后台 **插件** 列表中激活 LeAutoPost
3. 进入 **设置 → LeAutoPost设置** 进行配置

---

## 配置说明

### 基本设置

1. **启用自动发布**：勾选后插件开始工作
2. **发布频率（秒）**：两次发布之间的最小间隔，如 3600 表示 1 小时
3. **时间限制**：可选，限制仅在指定时间段内发布
4. **发布数量限制**：可选，限制最大发布数量

### 外部 Cron 设置

启用「使用外部 Cron」后，使用以下 URL 配置系统 Cron 或定时任务服务：

```
https://你的域名/?le_auto_post_cron=1&secret=你的密钥
```

- 密钥在设置页面自动生成，请妥善保管
- 建议设置为每小时执行一次，例如：`0 * * * *`（每小时整点）

### 内容过滤

- **过滤关键词**：每行输入一个需要过滤的关键词
- **替换字符**：替换后的字符，留空则删除关键词
- **内容过滤选项**：按需勾选过滤图片、链接或 URL

---

## 使用流程

1. 在 WordPress 后台创建文章并保存为 **草稿**
2. 在 **设置 → LeAutoPost设置** 中启用自动发布并完成配置
3. 插件将按设定规则自动发布最早的草稿
4. 可通过「立即发布一篇草稿」按钮手动触发发布
5. 在「发布日志」区域查看运行记录

---

## 系统要求

- WordPress 4.5.0 或更高版本
- PHP 5.6 或更高版本
- 已测试至 WordPress 6.7.2
